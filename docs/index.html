Project Overview

SkolarD is a tutoring platform built as a final-year capstone project for COMP 3350.

Vision

To create an accessible and intuitive tool for students to find and work with tutors.

Intended Users

Students and tutors in academic institutions.

Key Features

  • Session booking and unbooking
  • Messaging system
  • Rating and feedback
